# 🖼️ Стохастический растр (Stochastic Halftone Processor)

Веб-инструмент для преобразования растровых изображений в чёрно-белый стохастический (FM) растр. Работает полностью в браузере с использованием алгоритма диффузии ошибок Флойда-Стейнберга. Не требует серверной части, зависимостей или установки.

## ✨ Возможности
- 📤 **Удобная загрузка**: Drag & Drop или выбор файла через системный диалог
- 📐 **Масштабирование**: Изменение размера всего изображения с опцией фиксации пропорций
- ✂️ **Интерактивное кадрирование**: Выделение области мышкой + автоматическое масштабирование до заданных пикселей
- 🎚️ **Порог яркости в реальном времени**: Автоматическая перерисовка при движении ползунка (с защитой от фризов)
- 💾 **Экспорт**: Сохранение результата в формате PNG
- 🔒 **Приватность**: 100% клиентская обработка. Изображения никогда не покидают ваш браузер
- ⚡ **Производительность**: Оптимизированная работа с `ImageData`, изоляция исходных пикселей для многократной перерисовки

## 🚀 Быстрый старт
1. Скачайте или скопируйте код в файл `index.html`
2. Откройте файл в любом современном браузере
3. **Порядок работы**: 
   Загрузить → (опционально) Масштабировать → Выделить область → Кадрировать → Подвигать ползунок порога → Скачать результат

## 🛠 Как это работает
Алгоритм преобразует цветное изображение в оттенки серого, а затем применяет **диффузию ошибок Флойда-Стейнберга**:
1. Каждый пиксель сравнивается с заданным порогом и заменяется на `0` (чёрный) или `255` (белый)
2. Ошибка квантования распределяется на соседние ещё не обработанные пиксели с весами `7/16, 3/16, 5/16, 1/16`
3. Результат — стохастическое распределение точек без регулярной сетки, имитирующее аналоговую печать или электронные чернила

Для стабильной работы при изменении порога используется изолированный буфер исходных пикселей (`preHalftoneData`), что предотвращает накопление артефактов при многократной перерисовке. Ползунок обрабатывается с `debounce` (150мс) для сохранения плавности интерфейса.

## 💻 Технические детали
| Компонент | Реализация |
|-----------|------------|
| **Язык** | Vanilla JS (ES6+) |
| **Графика** | Canvas 2D API, `ImageData`, `Uint8ClampedArray` |
| **Загрузка** | `FileReader`, Drag & Drop API |
| **Оптимизация** | Глубокое копирование буфера, `requestAnimationFrame`, debounce слайдера |
| **Зависимости** | Отсутствуют. Работает автономно из одного `.html` файла |
